# Portfolio Blog

This is a porfolio/blog site using a next.js template from Vercel.

Includes:

-   MDX and Markdown support
-   Optimized for SEO (sitemap, robots, JSON-LD schema)
-   RSS Feed
-   Dynamic OG images
-   Syntax highlighting
-   Tailwind v4
-   Vercel Speed Insights / Web Analytics
-   Geist font
